## 项目介绍
以慕课网的SpringBoot商城项目为基础学习商城项目的后端开发
Spring Boot + MyBatis
项目重难点：
1. 递归获取目录列表，利用redis缓存返回目录结果
2. 利用mybatis-generator生成dao层
3. aop统一处理web请求日志
4. 设置统一相应对象、统一异常处理对象
5. 登陆的过滤器（包括用户的 管理员的）
6. 过滤器中统一将用户session对象进行了保存（保存在线程中）
7. 请求参数校验、swagger接口文档
8. 实现图片上传，uuid重命名，设置静态文件映射目录
8. 商品排序、排序（字段使用枚举）、关键词搜索、模糊查找功能的实现
9. 购物车的返回非标准对象（表链接查询）
10. 支付二维码生成

存在不足：
购物车实时检查库存等
